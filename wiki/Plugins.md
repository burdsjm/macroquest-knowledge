# Plugins

## Plugin catalog template

| Plugin | Purpose | Required? | Key Commands | Notes |
|---|---|---|---|---|
| plugin-name | short description | yes/no | `/plugin ...` | caveats |

## Per-plugin subpage convention

Create one page per major plugin, for example:

- `Plugin-MyPlugin`

Recommended sections:

1. Overview
2. Install/load
3. Configuration
4. Common commands
5. Troubleshooting
6. Compatibility notes

## Operational tips

- Load only what you need.
- Note startup dependencies/order.
- Track plugin-specific crashes in [[Troubleshooting]].
