# Glossary

- **Aggro**: Enemy threat toward player/group.
- **Alias**: Shorthand command mapped to one or more MQ commands.
- **Bind**: Keyboard mapping for command execution.
- **Macro**: Script written in MQ macro language.
- **Module**: Reusable Lua/script component.
- **Plugin**: Compiled extension for MQ.
- **State machine**: Pattern using explicit states and transitions.
- **TLO**: Top-level object used to query game/runtime data.
