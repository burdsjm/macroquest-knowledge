# Common Snippets

Copy/paste-ready examples with short explanations.

## Suggested snippet categories

- Target checks
- Range/path safety checks
- Cast retry logic
- Assist logic
- Follow/camp recovery
- Simple state machine skeleton

## Snippet quality rules

- Include assumptions at top.
- Include stop condition.
- Keep examples minimal and composable.
- Link to full docs for each command/TLO used.
