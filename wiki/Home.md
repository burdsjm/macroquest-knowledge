# MacroQuest Knowledge Wiki

Welcome to the community-maintained MacroQuest knowledge base.

This wiki is structured to mirror common learning paths from the official docs at [docs.macroquest.org](https://docs.macroquest.org), while adding practical usage notes and templates for day-to-day play.

## Quick start for repository admins

To enable the GitHub Wiki for this repository:

1. Open **Settings → General → Features**.
2. Ensure **Wikis** is checked.
3. Open the **Wiki** tab.
4. Click **Create the first page**.
5. Copy/paste pages from this `wiki/` directory into the GitHub Wiki (or push them to the `*.wiki.git` repository).
