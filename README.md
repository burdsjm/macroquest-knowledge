# macroquest-knowledge

This repository hosts a structured MacroQuest knowledge base starter built from and aligned to the official documentation at [docs.macroquest.org](https://docs.macroquest.org).

## Wiki bootstrap

A ready-to-publish GitHub Wiki scaffold is included under [`wiki/`](wiki).

To publish:

1. Enable **Settings → General → Features → Wikis** in GitHub.
2. Open the repo **Wiki** tab and create the first page.
3. Copy pages from `wiki/` into the GitHub Wiki (or push to the repo's `.wiki.git`).
